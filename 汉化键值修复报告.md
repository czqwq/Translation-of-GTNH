## 四、验证方法

修复完成后，可用以下命令验证不再存在旧式废弃 key：

```bash
# 应无输出
grep -r "^gt\.component\." resources --include="*.lang"
grep -r "^gtnop\.world\.[A-Z]" resources --include="*.lang"
grep -r "^ig\.elevator\.structure\.AnyBaseCasingWith[0-9]Dot" resources --include="*.lang"
grep -r "^gtnhlanth\.tt\.hintdot" resources --include="*.lang"
grep -r "^BW\.infoData\.htgr\." resources --include="*.lang"
grep -r "^interaction\.separateBusses\." resources --include="*.lang"
```
